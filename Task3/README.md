## stdin parser
Программа получает на вход последовательность типа слово+разделитель, например, word1||word2>> и так далее, где ||, >> есть специальные слова разделители. Таких слов разделителей имеется >>, ||, &&, <, >, &, |. После ввода последовательности, разделённые слова выводятся в отсортированном виде в stdout.
## основные структуры данных
wordlist.h содержит функции для работы со структурами данных wordlist и wordbuf и их определения. wordlist содержит структуру список строк типа char ** , где будут содержаться накопленные из stdin слова. wordbuf структура данных, накапливающая слово из stdin.
