## shell commands
Частично реализованы команды shell Linux: echo, pwd, ls, cp, grep, tail.
## echo
Для команды echo реализованы флаги -s и -n, первая убирает пробелы между словами, вторая убирает символ '\n' перехода на новую строку.
## pwd
Печатает путь текущей рабочей директории
## ls
Для этой команды реализованы флаги -R, -l, -g. -R рекурсивно проходит по содержащимся каталогам и печатает их содержимое, так же печатаются файлы, лежащие в текущей директории. -l печатает аттрибуты файлов. -g выводит имя группы владельца.
## cp
cp file1 file2. Копирует содержимое file1 в file2.
## grep
Для этой команды реализованы флаги -v и -s, -v печатает все строки, в которых не встречается заданная подстрока. -s указывает номера строк, найденных программой в файле.
## tail
Для этой команды реализованы флаги -n и +n. -n печатает n строк с конца, +n печатает с n строки и до конца файла.
